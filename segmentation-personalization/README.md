# Segmentation & Personalization

## Overview

This project focuses on behavioral clustering and customer targeting models aimed at driving personalized marketing campaigns. By analyzing customer purchase behavior, promo sensitivity, and engagement metrics, we create actionable customer segments to optimize campaign effectiveness.

## Features

- Data preprocessing and feature engineering for customer behavior  
- Dimensionality reduction using PCA for better visualization  
- Behavioral clustering using K-Means algorithm  
- Rule-based offer matching engine to assign personalized promotions  
- Visualization of customer segments and campaign targeting
