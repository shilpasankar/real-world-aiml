# Airport Recommendation System - Demand Forecasting

## Overview

This project implements a demand forecasting system for airport retail using time series analysis. The goal is to predict passenger footfall and sales to optimize inventory, staffing, and promotions.

## Features

- Time series data processing and visualization  
- Demand forecasting using models like ARIMA and Prophet  
- Performance evaluation with metrics like MAE, RMSE  
- Modular code for easy adaptation to different airports or retail categories
